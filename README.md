# SpotITI: Indoor Navigation

SpotITI is an advanced indoor navigation system developed to facilitate real-time indoor positioning and navigation.

## Key Features

- **Real-Time Indoor Positioning**: Utilizes QR code scanning and accelerometer data to provide real-time indoor positioning and navigation for the users.
- **Compass View**: A compass view that responds to the device's orientation, displaying the user's current direction within the institute.
- **Searchable List of Rooms**: A searchable room list enables users to quickly find and navigate to their desired locations.

## Technologies

- **SwiftUI**: Used for building the user interface.
- **Core Location**: Implemented for indoor positioning.
- **AVFoundation**: Employed for QR code scanning.
- **Core Motion**: Used for determining device orientation.

## Installation

To run the application, clone this repository and open the project in Xcode. Make sure to have Xcode installed and updated to the latest version. 
The backend code is available in [this repository](https://github.com/GioIacca9/BussolaITIBackend).
